This project demonstrates a beginner-friendly *data analysis workflow* using Python, Pandas, NumPy, and Excel.  
It analyzes simulated sales data to uncover key trends, automate reporting, and derive actionable business insights.


 Key Highlights
 Processed and analyzed sales data using *Pandas* and *NumPy*.  
 Performed *data cleaning, aggregation, and performance analysis* by region and product.  
 Automated report generation and exported final results to *Excel*.  
 Delivered a simple yet effective analytics solution showcasing *data handling and business intelligence skills*.



## Tech Stack
Python | Pandas | NumPy | Excel
